## Workflow Piloto (Echo-Pilot)
- **Estado:** PENDIENTE MANUAL.
- Se intentó crear mediante API en n8n (PowerShell), pero no se pudo porque `N8N_API_KEY` contiene el valor de placeholder (`PLACEHOLDER_REEMPLAZAR`).
- **Acción requerida por el usuario**:
  1. Ir a n8n UI.
  2. New workflow → Nombre: `Shinobi Echo Pilot`
  3. Nodo Webhook: POST a path `shinobi-echo`.
  4. Nodo Set: Añadir `origin='n8n'` y `echoed_at={{$now}}`.
  5. Nodo Respond to Webhook: devolver el JSON completo.
  6. Activar el Workflow.
