## Issues observadas y solucionadas

### Falsos failures en CI

El step "Stop early if tag already exists" usaba `&& exit 0` que en GitHub Actions colorea el run como red incluso cuando el comportamiento es correcto. Reemplazado por un `Note tag already exists` step puramente informativo. Los `if: tag_exists == 'false'` en los demás steps ya garantizan que no se ejecute nada cuando el tag ya existe.

### Pages deploy failures

El workflow `pages.yml` no logra deploy porque GitHub Pages requiere repo público. Ya está documentado en `docs/manual_actions.md` — se desbloquea con E2.
