1. Instalar el ejecutable

Shinobi se distribuye como un único .exe para Windows x64. No necesitas Node, Python ni dependencias previas.

  1. Descarga shinobi.exe desde la sección Releases del repo.
  2. Cópialo a una carpeta tuya, por ejemplo C:\Users\tunombre\Shinobi\.
  3. Abre PowerShell o CMD en esa carpeta y ejecuta shinobi.exe.
  4. La primera vez verás un asistente que te pedirá idioma, API key y carpeta de memoria.

Si Windows SmartScreen bloquea el archivo, pulsa "Más información" → "Ejecutar de todos modos". Aún no firmamos el binario.

2. Pedir tu API key

Shinobi puede funcionar 100% local con tu propia clave de OpenAI u OpenRouter. Si quieres usar OpenGravity (cloud), necesitas además una clave de Zapweave.

  1. Apúntate en la lista de espera.
  2. Cuando entres en la beta, recibes un email con tu SHINOBI_API_KEY.
  3. El asistente del primer arranque la pedirá; si ya está corriendo, escribe shinobi config set api_key <tu-key>.

3. Comandos CLI

Comandos básicos (los más usados primero):

shinobi                          # arranca el modo interactivo
shinobi "investiga canales de productividad en inglés con +500k subs"
shinobi mode local               # solo local, ignora OpenGravity
shinobi mode kernel              # delega misiones grandes a OpenGravity
shinobi memory show              # ver memoria persistente
shinobi memory clear             # limpiar memoria
shinobi config show              # ver configuración actual
shinobi config set api_key XXX
shinobi --help                   # ayuda completa

Mientras estés dentro del modo interactivo puedes pulsar Ctrl+C para abortar la respuesta en curso y exit para cerrar.

4. Preguntas frecuentes

¿Mis ideas y búsquedas se suben a la nube?

No por defecto. Shinobi es local: la memoria, los prompts y las transcripciones viven en tu PC. Solo se envía a un LLM externo el texto que ese paso requiere razonar, y solo si has configurado un proveedor de LLM. Si activas OpenGravity (plan Pro / Founders), las misiones cloud sí se ejecutan en nuestra infraestructura, pero tus archivos locales nunca se sincronizan automáticamente.

¿Qué LLMs usa Shinobi?

Por defecto OpenAI gpt-4o para razonamiento y gpt-4o-mini para tareas baratas. Para visión, OpenRouter con google/gemini-2.0-flash por coste. Puedes cambiar el proveedor en config; soportamos cualquier endpoint compatible con la API de OpenAI.

¿Cómo se cuentan los créditos OpenGravity?

1 crédito ≈ una invocación pequeña de modelo. Las misiones de investigación profunda consumen entre 50 y 500 créditos según duración y modelos elegidos. Los planes Founders y Pro incluyen una cuota mensual amplia para uso normal de un creador; si la superas, puedes recargar paquetes adicionales o esperar al siguiente ciclo. La cuota concreta se confirma cuando OpenGravity sale de beta.

¿Cómo cancelo mi suscripción?

Desde tu panel en zapweave.com (botón "Cancelar plan"), o escribiendo a hola@zapweave.com. La cancelación es inmediata y no hay penalización; mantienes el acceso hasta que termine el ciclo ya pagado. Si eras Founders y vuelves más tarde, no se garantiza recuperar el precio bloqueado.

¿Qué pasa con mis datos si dejo de pagar o cierro la cuenta?

Tu Shinobi local sigue funcionando: la memoria, los reportes y las plantillas son archivos en tu PC. Lo único que dejará de funcionar es OpenGravity (las misiones cloud) si tu suscripción no está activa. Si pides borrado de cuenta, eliminamos tus datos cloud en un máximo de 30 días.
