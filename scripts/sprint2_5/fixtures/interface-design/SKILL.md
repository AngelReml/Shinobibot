# Interface Design

Activate when the user asks for UI mockups, wireframes, or component specs.

Steps:
1. Ask for target platform.
2. Identify the primary user action.
3. Apply 60-30-10 color rule.
4. Produce textual wireframe with ASCII boxes.
5. List accessibility considerations.
